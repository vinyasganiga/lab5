# Simple Calculator

A small responsive calculator built with HTML, CSS, and JavaScript.

Features:
- Basic operations: + − × ÷
- Decimal input and percent
- Clear and backspace
- Chaining of operations
- Keyboard support
- Simple handling for division by zero and floating-point rounding

Files:
- index.html — UI and structure
- styles.css — styling
- script.js — calculator logic

Usage:
Open index.html in a browser.
